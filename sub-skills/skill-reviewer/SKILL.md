# 🔍 Skill Reviewer - 评测技能质量

## 四阶段流程

```
扫描技能 → 质量评估 → 测试验证 → 改进建议
```

---

## 阶段一：扫描技能

### 确定评测对象

| 用户输入 | 目标 |
|----------|------|
| "评测 xxx 技能" | 指定技能 |
| "测试一下这个" | 当前对话提到的 |
| "整理技能" | 全部技能（简化评估） |

### 提取技能信息

```bash
# 扫描技能目录
ls -la /app/working/workspaces/default/skills/<skill-name>/
ls -la /app/working/skill_pool/<skill-name>/

# 读取核心文件
read_file("SKILL.md")
```

### 提取评估数据

| 数据项 | 来源 |
|--------|------|
| name | frontmatter |
| description | frontmatter |
| 正文行数 | wc -l SKILL.md |
| 子目录 | ls 结果 |
| 触发词 | description |

---

## 阶段二：质量评估 ⚡

### 评分维度

| 维度 | 权重 | 评分标准 | 满分 |
|------|------|----------|------|
| **触发准确性** | 20% | 触发词明确、无歧义 | 10 |
| **描述质量** | 20% | description 清晰完整 | 10 |
| **结构完整性** | 20% | 有必要目录和文件 | 10 |
| **流程清晰度** | 20% | 步骤明确、可执行 | 10 |
| **边界处理** | 10% | 有异常处理和边界条件 | 10 |
| **文档完善度** | 10% | 有示例、路径说明 | 10 |

### 评分等级

| 总分 | 等级 | 标识 | 建议 |
|------|------|------|------|
| 9-10 | 优秀 | 🟢 A | 保持现状 |
| 7-8.9 | 良好 | 🟢 B | 小优化 |
| 5-6.9 | 一般 | 🟡 C | 需要改进 |
| 3-4.9 | 较差 | 🟠 D | 重构建议 |
| < 3 | 差 | 🔴 F | 不推荐使用 |

### 评估输出

```
📋 **质量评估报告**

**技能**：skill-name
**版本**：v1.0.0
**评估时间**：2026-04-18

─────────────────────────────────────
**综合评分**：8.5 / 10 🟢 B

| 维度         | 得分 | 满分 | 评价          |
|--------------|------|------|---------------|
| 触发准确性   | 9    | 10   | 触发词明确    |
| 描述质量     | 8    | 10   | 描述完整      |
| 结构完整性   | 8    | 10   | 结构合理      |
| 流程清晰度   | 8    | 10   | 步骤清晰      |
| 边界处理     | 7    | 10   | 有基本处理    |
| 文档完善度   | 9    | 10   | 示例充分      |

─────────────────────────────────────

**优势**：
- [优势1]
- [优势2]

**待改进**：
- [改进点1]
- [改进点2]
```

## 阶段三：测试验证

### 测试场景设计

| 场景类型 | 描述 | 预期结果 |
|----------|------|----------|
| **核心场景** | 典型使用场景 | 正常执行 |
| **边界场景** | 边界条件输入 | 正确处理 |
| **异常场景** | 异常/错误输入 | 优雅失败 |

### 执行测试

在对话中模拟用户输入，验证技能响应。

### 记录结果

```
🧪 **测试结果**

| 场景       | 输入           | 预期         | 实际        | 结果  |
|------------|----------------|--------------|-------------|--------|
| 核心场景   | "触发词"       | 执行流程     | 执行流程    | ✅ PASS |
| 边界场景   | "空输入"       | 优雅提示     | 报错        | ❌ FAIL |
```

## 阶段四：改进建议

### 优先级排序

| 优先级 | 问题类型 | 建议 |
|--------|----------|------|
| P0 | 安全风险 | 立即修复 |
| P1 | 核心功能缺失 | 尽快改进 |
| P2 | 体验问题 | 可选优化 |
| P3 | 文档问题 | 低优先级 |

### 改进模板

```markdown
## 📝 改进建议

### P1：描述优化
**问题**：description 不够清晰
**建议**：[具体建议]
**示例**：[代码示例]

### P2：结构优化
**问题**：缺少必要目录
**建议**：[具体建议]
```

## 执行命令

```bash
# 扫描技能信息
wc -l /app/working/workspaces/default/skills/<skill>/SKILL.md

# 列出目录结构
find /app/working/workspaces/default/skills/<skill>/ -type f
```
