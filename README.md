## <a id="section-zh"></a>🇨🇳 中文说明

### 简介

CoPaw 技能管理调度器。

### 文件结构

```
skill-manager/
├── LICENSE.txt
├── README.md
├── SKILL.md
```

### 安装

将 `skill-manager` 目录放入工作区的 `skills/` 目录下即可使用。

### 使用方式

详细用法请参考 `SKILL.md`。

### 依赖

- QwenPaw / CoPaw 环境
- Python 3.8+

### 子技能

| 子技能 | 说明 |
|--------|------|
| `skill-creator` | 从零创建 CoPaw 技能。 |
| `skill-editor` | 编辑/修改 CoPaw 技能。 |
| `skill-finder` | 查找/发现AI技能。 |
| `skill-healer` | 技能自愈系统。 |
| `skill-installer` | 安装外界技能到 CoPaw。 |
| `skill-merger` | 融合技能：将功能重叠或互补的技能合并生成新技能。 |
| `skill-reviewer` | 评测 CoPaw 技能质量。 |

## <a id="section-en"></a>🇬🇧 English Documentation

### Introduction

CoPaw 技能管理调度器

### 文件结构

```
skill-manager/
├── LICENSE.txt
├── README.md
├── SKILL.md
```

### 安装

Place the `skill-manager` directory into the `skills/` directory of your workspace.

### 使用方式

Refer to `SKILL.md` for detailed usage.

### 依赖

- QwenPaw / CoPaw environment
- Python 3.8+

### Sub-skills

| 子技能 | 说明 |
|--------|------|
| `skill-creator` | 从零创建 CoPaw 技能 |
| `skill-editor` | 编辑/修改 CoPaw 技能 |
| `skill-finder` | 查找/发现AI技能 |
| `skill-healer` | 技能自愈系统 |
| `skill-installer` | 安装外界技能到 CoPaw |
| `skill-merger` | 融合技能：将功能重叠或互补的技能合并生成新技能。四阶段流程：扫描分析 → 适合性评估 → 规模决策 → 确认执行。融合后自动标识来源技能。 |
| `skill-reviewer` | 评测 CoPaw 技能质量 |
